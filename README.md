# Instructor-Effectiveness-Modeling
Machine Learning project to analyze and Classify instructor effectiveness using Python, Pandas and Scikit-learn 

## Dataset
**Note** The dataset is not included in this repository because it was provided as part of an assessment and is not publicly shared.

## Objective
The objective of this project is to evalute instructor effectiveness using learner outcomes, engagement, and feedback metrics. A Decision Tree Classifier was used to classify instructors into low, Medium and High effectiveness tiers.

## Technologies Used
-Python
-Pandas
-Numpy
-Matplotlib
-Scikit-learn
-Google-Colab

## Project Workflow
- Data Loading
- Exploratory Data Analysis (EDA)
- Instructor-level Data Aggregation
- Feature Engineering
- Instructor Effectiveness Score Calculation
- Effectiveness Tier Creation
- Decision Tree Classification
- Model Evaluation
- Feature Importance Analysis

## Results
- Modal Accuray: **87.5%**
- Completion Rate and Dropout Rate were the most influential
- The model successfully classified instructors into Low, Medium and High effectiveness tiers.
